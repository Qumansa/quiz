## Тематика проекта:
Викторина по веб-разработке с тремя уровнями сложности.
___
## Технологии проекта:
+ React:
  + React Hooks;
  + react-router-dom.
+ Redux:
  + Redux Toolkit;
  + RTQ Query.
+ Sass;
+ Работа с сервером. В качестве сервера используется локальный json-server.
___
## Структура проекта:
- Главная;
- Викторина;
- Страница 404.
